# 肝炎個案管理追蹤開發需求

提案示範視圖以及說明文件，以協助開發功能建立。

---

## 專案說明

本專案為「肝炎個案追蹤」功能之提案與規格文件，預計建置於既有醫療系統平台上。內容包含：

- **模擬示範視圖**：以 HTML、CSS、JavaScript 建立的互動式 mock 介面
- **BDD 規格文件**：行為驅動開發規格，供開發與測試對照
- **資料定義**：病人資料、檢驗項目、影像檢查、追蹤碼等結構說明
- **開發規則**：前端技術規範與模擬原則

### 重要說明：檢驗數值基準

**所有檢驗數值皆以本院的單位與標準值／警示值的數值為基準。**

模擬文件中的參考值、單位、異常判斷邏輯，於實際開發時應改為串接本院檢驗系統之單位與標準值／警示值設定，以確保臨床顯示正確性。

## 快速開始

1. 以瀏覽器開啟 `mock/index.html`
2. 輸入身分證字號或病歷號（範例：`A123456789`、`MR20240001`、`B987654321`）
3. 點擊「查詢」檢視模擬功能

---

## 主要功能

| 功能 | 說明 |
|------|------|
| 病人檢索 | 以身分證字號或病歷號查詢，顯示基本資料 |
| 病人基本資料 | 75% 區塊顯示姓名、聯絡方式等 |
| FIB-4 計算 | 25% 區塊，輸入年齡/AST/ALT/PLT 計算肝纖維化風險 |
| 肝炎追蹤項目 | 篩選按鈕（一般肝炎、HBV、HCV）、可折疊分類、檢驗歷程 |
| 腹部影像報告 | 項目、醫令代碼、日期；點擊顯示報告內容 |
| 個案管理邏輯提示 | 依檢驗結果與代碼歷程顯示收案／追蹤建議 |
| 個案管理代碼開立歷程 | P4201、P4202 開立紀錄 |
