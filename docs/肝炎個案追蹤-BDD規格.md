# 肝炎個案追蹤 - BDD 規格文件

## 概述

本文件描述「肝炎個案追蹤」功能之行為驅動開發（BDD）規格，此功能將建置於既有醫療系統平台上。

---

## 功能一：病人資料檢索

### 情境：以身分證字號檢索病人

```gherkin
Feature: 病人資料檢索
  作為醫療人員
  我希望能夠透過身分證字號或病歷號檢索病人
  以便取得肝炎追蹤所需之病人基本資料

  Scenario: 以身分證字號成功檢索病人
    Given 系統已連線至病人資料庫
    And 病人資料庫中存在身分證字號 "A123456789" 的病人
    When 醫療人員於檢索欄位輸入身分證字號 "A123456789"
    And 點擊「查詢」按鈕
    Then 系統應從病人資料庫取得該病人基本資料
    And 應顯示以下欄位：
      | 欄位名稱 | 說明 |
      | 姓名 | 病人姓名 |
      | 性別 | 病人性別 |
      | 身分證號碼 | 病人身分證字號 |
      | 病歷號 | 病人病歷號 |
      | 生日 | 出生日期 |
      | 年齡 | 依生日計算之年齡 |
      | 地址 | 通訊地址 |
      | 連絡電話-手機 | 手機號碼 |
      | 連絡電話-住家 | 住家電話 |
      | 連絡電話-公司 | 公司電話 |
      | 初診日期 | 首次就診日期 |

  Scenario: 以病歷號成功檢索病人
    Given 系統已連線至病人資料庫
    And 病人資料庫中存在病歷號 "MR20240001" 的病人
    When 醫療人員於檢索欄位輸入病歷號 "MR20240001"
    And 點擊「查詢」按鈕
    Then 系統應從病人資料庫取得該病人基本資料
    And 應顯示完整基本資料欄位（同身分證字號檢索）

  Scenario: 檢索無資料之病人
    Given 系統已連線至病人資料庫
    And 病人資料庫中不存在身分證字號 "Z999999999" 的病人
    When 醫療人員於檢索欄位輸入身分證字號 "Z999999999"
    And 點擊「查詢」按鈕
    Then 系統應顯示「查無此病人資料」之提示訊息
```

## 功能二：肝炎追蹤項目區塊

### 情境：肝炎追蹤項目表格顯示

```gherkin
Feature: 肝炎追蹤項目
  作為醫療人員
  我希望能夠檢視病人的肝炎相關檢驗項目最新數值
  以便進行個案追蹤與評估

  Background:
    Given 已成功檢索並顯示病人基本資料
    And 病人資料庫包含該病人之檢驗紀錄

  Scenario: 區塊佈局為左50%右50%
    Given 醫療人員檢視肝炎追蹤區塊
    Then 左側 50% 應顯示表格
    And 右側 50% 預留給檢驗歷程或報告內容顯示

  Scenario: 顯示肝炎追蹤項目表格
    Given 病人 "王小明" 有肝炎相關檢驗紀錄
    When 醫療人員檢視肝炎追蹤區塊
    Then 應顯示標題為「肝炎追蹤項目」的表格
    And 表格應包含以下欄位：「項目」、「值」、「參考值」、「日期」、「檢驗歷程」
    And 項目欄位應顯示項目名稱與單位，例如 "AST (U/L)"
    And 應顯示以下檢驗項目及其最新一筆數值：
      | 項目（含單位） | 說明 |
      | AST (U/L) | 天門冬胺酸轉胺酶 |
      | ALT (U/L) | 丙胺酸轉胺酶 |
      | Total bilirubin (mg/dL) | 總膽紅素 |
      | Direct bilirubin (mg/dL) | 直接膽紅素 |
      | Albumin (g/dL) | 白蛋白 |
      | AFP (ng/mL) | 甲型胎兒蛋白 |
      | HBsAg | B型肝炎表面抗原 |
      | HBeAg | B型肝炎e抗原 |
      | Anti-HCV | C型肝炎抗體 |
      | HCV RNA | C型肝炎病毒量 |
      | PT (sec) | 凝血酶原時間 |
      | APTT (sec) | 活化部分凝血活酶時間 |
      | 腹部超音波 | 腹部超音波檢查結果 |

  Scenario: 值與參考值異常時顯示警示顏色
    Given 病人檢驗值超過參考值，或定性項目為陽性/有異常值
    When 醫療人員檢視肝炎追蹤區塊
    Then 該「值」應以警示顏色顯示（如紅色或橘色）
    And 檢驗歷程列表中的值若異常，亦應以警示顏色顯示

  Scenario: 腹部超音波項目顯示檢視報告
    Given 病人有腹部超音波檢驗紀錄
    When 醫療人員檢視肝炎追蹤區塊
    Then 腹部超音波項目的「值」欄位應顯示「檢視報告」
    And 點擊「檢視報告」後應於右側區塊顯示報告內容
    And 報告內容應以可捲動的文字框顯示

  Scenario: 檢驗項目無資料時顯示破折號
    Given 病人 "王小明" 的 AST 項目於資料庫中無檢驗紀錄
    When 醫療人員檢視肝炎追蹤區塊
    Then AST 項目的「值」欄位應顯示 "-"
    And AST 項目的「日期」欄位應顯示 "-"

  Scenario: 點擊檢驗歷程顯示歷程列表
    Given 病人 "王小明" 的 ALT 項目有多筆檢驗紀錄
    When 醫療人員點擊 ALT 項目的「檢驗歷程」欄位
    Then 應於區塊右側 50% 顯示歷程列表
    And 歷程列表應包含「值」、「參考值」與「日期」欄位
    And 歷程應依日期排序（最新在前或最舊在前，依業務需求）
```

## 功能三：腹部檢驗報告區塊

### 情境：腹部檢驗報告列表與內容顯示

```gherkin
Feature: 腹部檢驗報告
  作為醫療人員
  我希望能夠檢視病人的腹部檢驗報告列表及完整報告內容
  以便進行臨床評估

  Background:
    Given 已成功檢索並顯示病人基本資料

  Scenario: 區塊佈局為左50%右50%
    Given 醫療人員檢視腹部檢驗報告區塊
    Then 左側 50% 應顯示報告列表表格
    And 右側 50% 預留給報告內容顯示

  Scenario: 顯示腹部檢驗報告表格
    Given 病人 "王小明" 有腹部檢驗報告紀錄
    When 醫療人員檢視腹部檢驗報告區塊
    Then 應顯示標題為「腹部檢驗報告」的表格
    And 表格應包含以下欄位：「項目」、「日期」

  Scenario: 點擊報告列顯示報告內容
    Given 病人 "王小明" 有一筆腹部超音波報告，日期為 "2024-01-15"
    When 醫療人員點擊該筆報告列
    Then 應於區塊右側 50% 顯示報告內容
    And 報告內容應以可捲動的文字框（scroll textbox）顯示
    And 應顯示完整報告文字內容
```

## 功能五：個案管理邏輯提示

### 情境：依邏輯判斷顯示收案與追蹤建議

```gherkin
Feature: 個案管理邏輯提示
  作為醫療人員
  我希望系統能依檢驗結果與管理代碼歷程自動判斷並顯示建議
  以便適時進行收案或追蹤提醒

  Background:
    Given 已成功檢索並顯示病人基本資料
    And 個案管理邏輯提示顯示於「個案管理代碼開立歷程」區塊上方

  Scenario Outline: 建議收案提示 - 管理代碼歷程為空且檢驗陽性
    Given 個案管理代碼開立歷程為空白（無任何紀錄）
    And <檢驗條件>
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「個案未曾在本院收案，建議收案」

    Examples:
      | 檢驗條件 |
      | HBsAg 最近一筆為陽性 |
      | Anti-HCV 最近一筆為陽性 |
      | HCV RNA 有異常值 |
      | HBsAg 最近一筆為陽性 且 Anti-HCV 為陽性 |
      | HBsAg 最近一筆為陽性 且 HCV RNA 有異常值 |
      | Anti-HCV 為陽性 且 HCV RNA 有異常值 |

  Scenario: 建議收案 - 多條件皆符合
    Given 個案管理代碼開立歷程為空白
    And HBsAg 最近一筆為陽性
    And Anti-HCV 最近一筆為陽性
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「個案未曾在本院收案，建議收案」

  Scenario: 初次追蹤個案 - 建議返診（p4201，滿3個月）
    Given 最新歷程代碼為 "p4201"
    And 該代碼開立日期距今已滿 3 個月
    And 開立日期距今未超過 6 個月
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「初次追蹤個案，建議返診追蹤」

  Scenario: 初次追蹤個案 - 逾期追蹤（p4201，超過6個月）
    Given 最新歷程代碼為 "p4201"
    And 該代碼開立日期距今已超過 6 個月
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「初次追蹤個案逾期追蹤, 下次可從新收案時間為 (初次開立1年之後)」
    And 提示中應顯示具體日期（初次開立日期 + 1 年）

  Scenario: 長期追蹤個案 - 建議返診（p4202，滿6個月）
    Given 最新歷程代碼為 "p4202"
    And 該代碼開立日期距今已滿 6 個月
    And 開立日期距今未超過 1 年
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「長期追蹤個案, 建議返診追蹤」

  Scenario: 長期追蹤個案 - 逾期追蹤（p4202，超過1年）
    Given 最新歷程代碼為 "p4202"
    And 該代碼開立日期距今已超過 1 年
    When 醫療人員檢視個案管理邏輯提示區
    Then 應顯示：「長期追蹤個案於期追蹤, 下次可能新收案時間為 (最後一次開立1年之後)」
    And 提示中應顯示具體日期（最後一次開立日期 + 1 年）

  Scenario: 無提示 - 管理代碼歷程為空但檢驗皆陰性
    Given 個案管理代碼開立歷程為空白
    And HBsAg 最近一筆為陰性
    And Anti-HCV 最近一筆為陰性
    And HCV RNA 無異常值
    When 醫療人員檢視個案管理邏輯提示區
    Then 不應顯示「建議收案」之提示
```

## 功能六：提示優先順序與顯示規則

```gherkin
Feature: 提示顯示優先順序
  作為醫療人員
  我希望系統能正確處理多種條件同時符合時之提示顯示
  以便獲得最適切的建議

  Scenario: 提示優先順序說明
    Given 系統可能同時符合多項提示條件
    When 判斷應顯示之提示
    Then 應依以下優先順序顯示（高優先覆蓋低優先）：
      | 優先順序 | 條件 | 提示內容 |
      | 1 | 管理代碼歷程為空 且 (HBsAg陽性 OR Anti-HCV陽性 OR HCV RNA異常) | 個案未曾在本院收案，建議收案 |
      | 2 | 最新代碼 p4201 且 開立距今 > 6個月 | 初次追蹤個案逾期追蹤... |
      | 3 | 最新代碼 p4202 且 開立距今 > 1年 | 長期追蹤個案於期追蹤... |
      | 4 | 最新代碼 p4201 且 開立距今 >= 3個月 | 初次追蹤個案，建議返診追蹤 |
      | 5 | 最新代碼 p4202 且 開立距今 >= 6個月 | 長期追蹤個案, 建議返診追蹤 |
```

## 附錄：資料定義

### 病人基本資料欄位

| 欄位 | 型別 | 說明 |
|------|------|------|
| 姓名 | string | 病人姓名 |
| 性別 | string | 男/女 |
| 身分證號碼 | string | 中華民國身分證字號 |
| 病歷號 | string | 院內病歷號 |
| 生日 | date | 出生日期 |
| 年齡 | string/number | 依生日計算 |
| 地址 | string | 通訊地址 |
| 連絡電話-手機 | string | 手機號碼 |
| 連絡電話-住家 | string | 住家電話 |
| 連絡電話-公司 | string | 公司電話 |
| 初診日期 | date | 首次就診日期 |

### 肝炎追蹤檢驗項目

| 項目代碼 | 項目名稱（含單位） | 參考值 | 說明 |
|----------|-------------------|--------|------|
| AST | AST (U/L) | 0-40 | 天門冬胺酸轉胺酶 |
| ALT | ALT (U/L) | 0-40 | 丙胺酸轉胺酶 |
| Total bilirubin | Total bilirubin (mg/dL) | 0.1-1.2 | 總膽紅素 |
| Direct bilirubin | Direct bilirubin (mg/dL) | 0-0.3 | 直接膽紅素 |
| Albumin | Albumin (g/dL) | 3.5-5.5 | 白蛋白 |
| AFP | AFP (ng/mL) | 0-20 | 甲型胎兒蛋白 |
| HBsAg | HBsAg | 陰性 | B型肝炎表面抗原，陽性為異常 |
| HBeAg | HBeAg | 陰性 | B型肝炎e抗原，陽性為異常 |
| Anti-HCV | Anti-HCV | 陰性 | C型肝炎抗體，陽性為異常 |
| HCV RNA | HCV RNA | 無異常 | C型肝炎病毒量，有異常值為異常 |
| PT | PT (sec) | 10-14 | 凝血酶原時間 |
| APTT | APTT (sec) | 25-35 | 活化部分凝血活酶時間 |
| 腹部超音波 | 腹部超音波 | - | 值顯示「檢視報告」，點擊顯示報告內容 |

### 個案管理代碼

| 代碼 | 說明 |
|------|------|
| p4201 | 初次追蹤個案（追蹤期限：3個月建議返診，6個月逾期） |
| p4202 | 長期追蹤個案（追蹤期限：6個月建議返診，1年逾期） |
